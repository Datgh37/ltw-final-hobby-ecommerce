B0: Khởi tạo app password trên trang myaccount.google/apppassword (login = mail sinh viên) để lấy được chuỗi ký tự mã

B1: Khởi tạo id user secret trong project
dotnet user-secrets init

B2: Lưu thông tin smtp (host và port đã có rồi nên chỉ cần lưu email gửi và app password), nên dùng mail sinh viên để gửi otp cho dễ test:

dotnet user-secrets set "MailSettings:Email" "your-email@gmail.com" 

dotnet user-secrets set "MailSettings:Password" "your-app-password-here"
